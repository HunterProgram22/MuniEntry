remove_db_connection(connection_name: str) → None¶

Removes database connection from application.

If a connection is removed it would need to be created again to be used again. This should only be used upon close of the application to clean up connections.

Parameters

connection_name (str) – A string set to identify the database connection.

set_server_and_database(connection_name: str) → tuple¶

Sets the server and database name for the SQL Server connections.

This function is used to set a local instance of the database for Justin to test at home without being connected to the delcity network.

Parameters

connection_name (str) – The name of the database connection.

Returns

(server, database)

server (str): The name of the server for the database connection.

database (str): The name of the database.

Return type

tuple
